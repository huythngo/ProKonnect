
----------Install the project------------
npm install
npm install --save-dev

----------Run the project-----------------
-Run the whole project (front-end && back-end)
npm run dev

-Run the server
npm run server

-Run the front-end
npm run client

After starting the project, 
Now the server will be up on port 5000 on your local machine. The front-end page will be up on port 3000.

-----------Test the project---------------

To run the test suites:
npm run test
